Maintient d'un code propre

Tout au long de son écriture, je cherche à optimiser mon code et à le rendre le plus facilement maintenable.

Cela peut permettre de diminuer drastiquement les temps de chargement et de faire des économies de temps lors de la modification du site.

Dans le cas de cette page, j'utilise un maximum les capacités de Bootstrap pour éviter de surcharger le style via SCSS. Mon code SCSS tient en 12 petites lignes !

Et pour un plus gros projet ?

Dans le cas de plus gros projets, j'utilise les propriétés que m'offrent SCSS pour gagner du temps et alléger le code.
Les @mixin , les @include et les variables peuvent être de redoutables alliées !
